dtf-generator-maven-plugin Maven Mojo

dtf-generator:build-layout

Full name:

com.dextratech.dtf:dtf-generator-maven-plugin:1.0-SNAPSHOT:build-layout

Description:

Goal que ayuda en la tarea de generar una plantilla a partir de un HTML generado por Selenium agregando wildcards del estilo de Velocity.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: process-sources.

Required Parameters

Name Type Since Description
finalName String - The final file name of the html layout
htmlFile File - Location of the HTML test case file generated by the Selenium IDE.
target String - Location of the /target folder

Optional Parameters

Name Type Since Description
verbose boolean - Define if this will to print logs in system console

Parameter Details

finalName:

The final file name of the html layout
  • Type: java.lang.String
  • Required: Yes
  • Expression: layout.html

htmlFile:

Location of the HTML test case file generated by the Selenium IDE.
  • Type: java.io.File
  • Required: Yes

target:

Location of the /target folder
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${project.build.directory}

verbose:

Define if this will to print logs in system console
  • Type: boolean
  • Required: No
  • Expression: true
