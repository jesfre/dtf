dtf-generator-maven-plugin Maven Mojo

dtf-generator:generate-tests

Full name:

com.dextratech.dtf:dtf-generator-maven-plugin:1.0-SNAPSHOT:generate-tests

Description:

Class used to generator test suites in base of a data set from a Excel file and a layout

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: process-sources.

Required Parameters

Name Type Since Description
excelDataSource String - Location of the XLS data source file for this test case script
layoutDirectory String - The file path of the html layout
testSuitesDirectory File - The directory path to save the generated tests

Optional Parameters

Name Type Since Description
verbose boolean - Define if this will to print logs in system console

Parameter Details

excelDataSource:

Location of the XLS data source file for this test case script
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${basedir}/src/test/resources/datasets/dataset.xls

layoutDirectory:

The file path of the html layout
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${project.build.directory}/layout/

testSuitesDirectory:

The directory path to save the generated tests
  • Type: java.io.File
  • Required: Yes
  • Expression: ${project.build.directory}/testsuites

verbose:

Define if this will to print logs in system console
  • Type: boolean
  • Required: No
  • Expression: true
